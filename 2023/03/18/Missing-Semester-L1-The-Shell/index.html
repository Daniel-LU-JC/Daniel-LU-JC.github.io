Missing Semester - L1 - The Shell

Posted on 2023-03-18 | In Missing Semester
several basic commands using shell

Environment Variables

a way to determine where a program is located (for shell)

the command will present a list of paths

Paths

the command will return the current working directory

navigate to the directory you were previously in

most programs implement this. ‘flag’ means anything that doesn’t take a value, or it is called an ‘option’.

the first command is used to change the location and the name of a file; the second one is ‘copy’

the first command is to remove an empty directory while the second one means remove recursively

Rewiring

rewiring by ‘<’ and ‘>’

rewire the input or output stream; last command shows another way to copy

append rather than overwrite

rewiring by pipe ‘|’

semantics: take the output of the program to the left and make it the input of the program to the right

pipe can be used for textual data, images, videos and so on. (to be introduced later)

Other Handy Tools

check manual pages

Also, ‘Ctrl + L’ is a handy keyboard shortcut to clear the terminal.

do the following things as the super user; second command means switch into a root terminal, which is indicated by a symbol ‘#’.

‘tee’ command takes its input and writes it to a file and also the terminal.

open the file in the appropriate program.
